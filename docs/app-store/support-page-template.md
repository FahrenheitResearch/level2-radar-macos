## About The App

`Macdar` is an iPhone radar app focused on fast live weather radar viewing.

## Troubleshooting

If radar data does not appear:
- Confirm the device has an active internet connection.
- Try switching to another station.
- Use the refresh control in the app.
- Close and reopen the app if the network feed was unavailable during launch.

If the app feels stale:
- Refresh the current station list.
- Confirm the selected product is available for the active station.

## Data Sources

Radar data is sourced from public NEXRAD Level II feeds and active NWS alert sources.
